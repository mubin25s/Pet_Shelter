## 🚀 How to Run Locally
1. Install **XAMPP**  
2. Place the project folder in `htdocs`  
3. Configure **Supabase credentials** in PHP config files  
4. Start Apache & MySQL (XAMPP)  
5. Open the project in your browser  

> Example: `http://localhost/pet_shelter/index.php`  

---

## 🔄 How It Works
1. Users browse the pet listings by category  
2. Click a pet to view its **detailed profile**  
3. Submit an **adoption request** (stored in Supabase)  
4. Admin logs in to **manage pets and requests**  
5. All changes are reflected dynamically through **PHP + Supabase**
